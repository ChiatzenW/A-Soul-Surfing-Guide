# 为了你我要

## 变体

XX我真的好喜欢你啊，为了你我要xx

## 来源

来源于凤凰战士徐文楷一个[流传甚广的电牛子视频](https://www.youtube.com/watch?v=hA7qx5vmbR0)（由于原视频真的露出了牛子，所以b站都是删减版）
>李可欣，我真的好喜欢你啊，为了你，我要电牛子！ 赫赫（掏出手电筒），噢噢噢————，我爱你

b站有许多的模仿整活相关视频

## 语义

一种展现男性魅力的表达自身感情的方式
