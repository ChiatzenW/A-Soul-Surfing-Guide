# 天无二日

## 变体

天无二日，我的心中只有XX一个太阳

## 来源

《礼记·曾子问》：“天无二日，土无二王，家无二主，尊无二上。”《第五共和国》中总统府警卫室室长车智澈（天文学家）殴打下属时所称“对我来说，天无二日，只有总统一个太阳”。因此朴正熙也被称为“（唯一的）太阳”

## 语义

表达对某人的喜爱，加强语气，有唯粉倾向
