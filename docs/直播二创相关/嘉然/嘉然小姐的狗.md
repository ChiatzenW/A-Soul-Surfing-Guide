# 嘉然小姐的狗

## 变体

嘉然小姐的猫/陆战功能犬

## 来源

b站用户 [新户眠子](https://space.bilibili.com/11192497)（已注销）
在 [嘉然评论区的小作文](https://t.bilibili.com/472642570902204872)，
原文应该是受《电锯人》“我还是想做玛奇玛小姐的狗” 的影响。

## 语义

表达对嘉然小姐的喜爱，或者单纯的图一乐
