# SC

## 变体

superchat

## 来源

在YouTube的直播上，可以通过打赏主播发送醒目留言(superchat)，缩写为sc。根据打赏金额的不同，sc的颜色也不同，红sc需要一万日元以上。

## 语义

b站也采用了这个机制，1000元及以上即为红sc
