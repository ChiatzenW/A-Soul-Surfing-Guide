# 毕业

## 变体

塔照 越塔

## 来源

a-rts游戏（dot类，或者叫moba）里的术语，指玩家在敌方防御塔攻击范围内，对敌方英雄进行直接攻击

## 语义

进行关注vtuber的行为被称为“推”后，与游戏术语“推塔”联想，讨论vtuber背后的一些真实信息（例如中之人、前世等）被称为冲塔，多含贬义，与私生饭等话术连用
