# 毒唯

## 变体

wd

## 来源

来源于传统韩娱日娱指偶像团体中只喜欢一个成员且厌恶其他成员的粉丝，v吧吧友经常使用此语

## 语义

只喜欢自家正主，用不理智行为伤害其他艺人的狂热粉丝，即 “唯” 一，并且在唯的基础上还有 “毒”。毒唯的行为包含但不限于拉踩、抹黑、诬蔑、造谣、屠广场、刷黑词条、带节奏、撕逼。
由于Asoul及其的评论区的特殊性，嘉然领进门，发病靠个人，此名词一般不具攻击性（绝大都是RP）
