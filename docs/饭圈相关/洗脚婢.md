# 洗脚婢

## 变体

## 来源

指的是偶像组合里的底边人物（比如SNH48里的林忆宁），出镜率不高，基本上是人肉背景板。

## 语义

饭圈常用于粉丝互掐，指粉丝对方idol存在感低，只配给自己偶像洗脚。
ASoul这边由于粉丝一点都得不到中之人的真实信息，想论证谁谁谁是底边都无从下手，巧妇难为无米之炊，只好自己发散RP娱乐。
