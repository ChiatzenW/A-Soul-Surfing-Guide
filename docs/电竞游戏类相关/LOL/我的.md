# 我的

## 来源

电竞游戏或者体育竞技中由于自己的失误给己方队友造成的损失表示歉意

## 语义

大意是“我的锅”

## 应用

[贝拉 6.18 贝拉的舞蹈教室又又又来啦！](https://www.bilibili.com/video/BV1bq4y1L77P?p=4&t=55m51s)

>乃琳没跳舞想混过去，拉姐读弹幕“我才发现啊，你们早就发现了吗？你们也不早点跟我说”
>
>弹幕纷纷表示“我的我的”
>
>嘉然乃琳表示“他们说了你没看见”
